冯·诺依曼计算机结构

冯·诺依曼结构将计算机分为运算器、控制器、存储器、输入/输出设备5大部分，程序和数据共享同一存储空间。

📖 核心解释

冯·诺依曼结构将计算机硬件划分为5大部分：运算器、控制器、存储器、输入设备、输出设备。控制器和运算器集成为CPU。指令和数据存储在同一存储器中，共享同一总线，按顺序执行。这是目前通用计算机的基础架构。

💻 伪代码

// 冯·诺依曼结构：指令和数据共享同一存储空间、同一总线
CPU <-> 总线 <-> 存储器(程序+数据混合存放)
输入设备 -> 存储器 -> 运算器 -> 输出设备
        ↑
      控制器(发出控制信号)

📌 实际例子

  • 现代PC（x86架构）均采用冯·诺依曼结构
  • Intel/AMD处理器属于典型的冯·诺依曼体系

✅ 优点

  • 结构简单易实现
  • 程序可被修改，灵活性强
  • 通用性强，适合各类计算任务

❌ 缺点

  • 存在冯·诺依曼瓶颈：CPU与存储器间带宽受限
  • 指令与数据共享总线，不能同时取指和取数，吞吐量受限
