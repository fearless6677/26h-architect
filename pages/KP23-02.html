著作权法核心条款

著作权包括人身权（发表权、署名权、修改权、保护作品完整权）和财产权（复制权、发行权等13项），其中署名权、修改权、保护作品完整权的保护期不受限制。

// 著作权归属判定 —— 像 switch-case 一样分流

function getCopyrightOwner(work) {
    if (work.type === "委托作品") {
        // 有合同按合同，没合同归受托人
        return work.contract ? work.contract.owner : "受托人";
    }

    if (work.type === "职务作品") {
        if (work.利用单位物质技术条件) {
            // 特殊职务作品 → 单位享有（署名权仍归作者）
            return { 著作权: "单位", 署名权: "作者" };
        } else {
            // 一般职务作品 → 作者享有，单位优先使用
            return { 著作权: "作者", 单位: "优先使用权" };
        }
    }

    // 合理使用：13种情形，不经许可、不付报酬
    if (isFairUse(work)) return "无需许可";
}

// 保护期判定
const 保护期 = {
    署名权:     "永久 ♾️",
    修改权:     "永久 ♾️",
    完整权:     "永久 ♾️",
    发表权:     "终生 + 死后50年",
    财产权:     "终生 + 死后50年",  // 法人作品：首次发表后50年
};
核心：著作权分人身权和财产权两大类。 人身权中，署名权、修改权、保护作品完整权保护期不受限制； 发表权与财产权有保护期（自然人终生+死后50年）。 职务作品分一般（归作者，单位优先使用）和特殊（署名权归作者，其他归单位）。 委托作品无约定 → 归受托人。
📋 合理使用13种：个人学习、适当引用、新闻报道、课堂教学… ⚖️ 署名+修改+完整 = 永久保护 💡 委托没合同 → 受托人

✅ 备考优势

  • 高频考点，委托作品和职务作品归属是每年必考内容
  • 合理使用情形可归纳记忆（13种分类背诵）
  • 规律清晰：人身权 vs 财产权二分法

⚠️ 易错点

  • 著作权权利项目多（17项），容易混淆
  • 职务作品一般 vs 特殊的区分需仔细理解
  • 发表权属于人身权，但保护期与财产权相同

📝 考试关键词

职务作品 委托作品 合理使用 署名权 保护期不受限制 受托人享有
