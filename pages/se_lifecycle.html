系统工程生命周期与综合集成法

一个系统工程从出生到退休要走完整条生命周期线

📖 核心解释

系统工程生命周期包括探索研究、概念阶段、开发阶段、生产阶段、使用阶段、保障阶段和退役阶段7个阶段，目的是以有序且高效的方式建立满足利益攸关者需求的框架。生命周期方法有计划驱动方法、渐进迭代式开发、精益开发和敏捷开发。此外，钱学森提出的综合集成法将系统分为简单系统和巨系统，开放的复杂巨系统具有整体论、相互联系、有序性、动态等基本原则，以及开放性、复杂性、进化与涌现性、层次性和巨量性等性质。

💻 伪代码

系统工程生命周期 = {
 阶段: [探索研究, 概念阶段, 开发阶段, 生产阶段, 使用阶段, 保障阶段, 退役阶段],
 方法: [计划驱动方法, 渐进迭代式开发, 精益开发, 敏捷开发]
}

综合集成法(钱学森) = {
 分类: "简单系统 vs 巨系统",
 开放的复杂巨系统特征: [整体论, 相互联系, 有序性, 动态],
 性质: [开放性, 复杂性, 进化与涌现性, 层次性, 巨量性]
}

📌 实际例子

  • 考试真题：系统工程的7个生命周期阶段
  • 钱学森综合集成法：从整体论出发研究开放复杂巨系统
  • 涌现性——整体具有部分所没有的性质，如蚁群的智能
