盈亏临界点（保本点）分析

盈亏临界点（保本点）是收入=成本的状态，核心公式：盈亏临界点销售量=总固定成本/(销售单价-单位可变成本)

📖 核心解释

盈亏临界点（保本点）是项目收入和成本相等的状态，即不盈不亏的状态。核心公式：利润=(销售单价-单位变动成本)×销售量-总固定成本；盈亏临界点销售量=总固定成本/(销售单价-单位可变成本)；盈亏临界点销售额=总固定成本/(1-总可变成本/销售收入)。真题中常结合税率计算，需注意税后收入的处理。

💻 伪代码

// 盈亏临界点公式
// 利润 = (销售单价 - 单位变动成本) × 销售量 - 总固定成本
// 盈亏临界点销售量 = 总固定成本 / (销售单价 - 单位可变成本)
// 盈亏临界点销售额 = 总固定成本 / (1 - 总可变成本/销售收入)
//
// 真题示例:
// 销售价=2500元/台, 销售量=25000台, 固定成本=250万, 可变成本=4000万, 税率=16%
// 单位可变成本 = 4000/25000 = 0.16万 = 1600元
// 税后单价收入 = 0.25×(1-16%) = 0.21万
// 0.21N = 250 + 0.16N  =>  N = 5000台

📌 实际例子

  • 真题：电视机售价2500元，总销售量25000台，固定成本250万，可变成本4000万，税率16%，求盈亏平衡点。解：单位可变成本=1600元，0.21N=250+0.16N，N=5000台

✅ 优点

  • 直观反映项目盈利所需的最低业务量
  • 为项目决策提供量化依据

❌ 缺点

  • 公式假设线性关系，实际业务中价格和成本可能非线性变化
