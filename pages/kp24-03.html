图解法——将约束画在坐标系，找可行域，平移等值线求最优解。
1
画约束直线：将每个不等式取等号，在坐标系中画直线
2
定可行域：根据不等号方向，取各直线正确侧的交集区域
3
画等值线：令目标函数 N = 某常数，画直线
4
平移等值线：沿增大方向平移，最后一个接触可行域的点即最优解
5
求最优值：将最优点坐标代入目标函数，得极值
联立方程组法——取等号联立，求交点，验证可行性，比较目标函数值。
1
取等号：将所有约束不等式改为等式
2
联立求解：两两联立方程，求交点坐标
3
验证可行：代入原不等式，检查是否全部满足
4
比较目标值：所有可行解代入目标函数，取最大/最小
// 线性规划 —— 图解法思路

// 约束条件
constraints = [
    2x + 3y ≤ 14,    // 原料约束
    x ≤ 2,            // 甲产量上限
    y ≤ 3,            // 乙产量上限 (应为 y≤4)
    x ≥ 0, y ≥ 0      // 非负约束
]

// 目标函数
maximize N = 2x + 3y

// Step 1: 画约束直线 → 找可行域顶点
vertices = findIntersections(constraints)
// → (0,0), (2,0), (2,4), (1,4), (0,14/3)

// Step 2: 验证每个顶点是否满足所有约束
feasible = filter(vertices, v => allSatisfied(v, constraints))

// Step 3: 代入目标函数，找最大值
optimal = max(feasible, v => eval(2x+3y, v))
// → (1, 4), N = 14 ✓
📌 例1：生产优化问题
生产甲(x)、乙(y)产品
约束：2x+3y≤14, x≤2, y≤4 (原题应为 8x≤16, 3y≤12)
利润：N = 2x + 3y
求最大利润

图解法：画可行域 → 平移等值线 → 最优点 (1, 4)
联立法：联立 x=1, y=4 交点 → 验证可行 → N = 2(1)+3(4) = 14万元

✅ 最优解 x=1, y=4，最大利润 14 万元
📌 例2：练习题
约束：2x+3y≤30, x+2y≥10, x≥y, x≥5, y≥0
目标函数：min N = 2x + 3y

可行域顶点中寻找使 N 最小的点 →
在 x=5, y=2.5 处，N = 2(5)+3(2.5) = 17.5

✅ 极小值在 (5, 2.5) 处 = 17.5
核心：线性规划 = 线性约束 + 线性目标函数 + 求极值。 图解法适合2个变量（直观不丢解），联立法适合≥3个变量。 最优解一定在可行域的顶点或边界上取到。
📐 图解法（2变量） 🔗 联立方程组法（≥3变量） 📈 等值线平移 🏭 生产优化 ⚡ 约束多时 O(n) 优势

✅ 优点

  • 图解法直观，不丢失正解
  • 约束多时计算量仅为 O(n)
  • 最优解必在顶点/边界取得

❌ 缺点 / 易错点

  • 联立方程组法可能丢失正解（如最优解在坐标轴交点）
  • 未知数 ≥ 3 时图解法难度增大
  • 需注意非负约束 x≥0, y≥0
