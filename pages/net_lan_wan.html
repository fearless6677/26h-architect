网络分类（按覆盖范围）：
LAN(局域网)  → IEEE 802.3(以太网), 802.11(WiFi)
MAN(城域网)  → IEEE 802.6, 三层: 核心层+汇聚层+接入层
WAN(广域网)  → 通信子网+资源子网, 需路由器/网关

// 以太网帧长度规则
最小帧长 = 64字节 (防冲突，按检测冲突最长时间定)
最大帧长 = 1518字节

// WiFi标准演进
802.11b → 802.11g → 802.11n(200Mb/s) → 802.11ac(1Gb/s)

// 5G核心特征
服务化架构(SBA): 网络功能灵活定制、按需组合
网络切片: 单个物理网上切出多个逻辑网(不同业务)
核心：LAN(局域网)覆盖范围小，以太网是最常见的LAN技术(IEEE 802.3)，最小帧长64字节、最大1518字节。WAN(广域网)覆盖广，由通信子网和资源子网组成。WLAN用IEEE 802.11标准(WiFi)。5G的两大核心特征是服务化架构(SBA)和网络切片。
办公室内10台电脑通过交换机和网线组成LAN，使用以太网协议通信 手机通过WiFi(802.11ac)以1Gb/s速率流媒体播放视频 5G网络切片：同一张5G物理网络切出一个低时延切片给自动驾驶，一个大带宽切片给视频直播

✅ 优点

❌ 缺点

  • LAN覆盖范围有限，不适合跨地域通信
  • 以太网最小帧长限制了对极短数据传输的效率
  • 5G网络建设成本高，基站密度大，覆盖范围受限于基站部署
