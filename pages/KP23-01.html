知识产权的基本特性

知识产权具有无体性、专有性、地域性和时间性四大特征。

🎬 动画演示
💻 伪代码
知识产权（无形财产权）
👻

无体性

对象没有具体形体，是抽象财富，不同于有形的物权客体。

💡 例：软件著作权保护的是代码逻辑，而非光盘本身
🔒

专有性

除权利人同意或法律规定外，他人不得使用。类似垄断权。

💡 例：未经许可使用他人专利 = 侵权
🌍

地域性

只在授予权利的国家范围内受保护，跨国需另行申请。

💡 例：中国软件著作权要在美国保护，需依美国法律
⏳

时间性

超过法定期限后，知识产品成为社会共同财富。

💡 例：发明专利保护期20年，到期后任何人可免费使用
// 知识产权四大特性 —— 抽象的无形财产权

class IntellectualProperty {
    // (1) 无体性：对象是抽象财富，没有具体形体
    nature()    → "无形的抽象财产"

    // (2) 专有性：非经同意/法定，他人不得使用
    exclusive()  → "权利人独占，他人禁用"

    // (3) 地域性：仅在授权国范围内受保护
    territory()  → "一国授权，一国保护"

    // (4) 时间性：法定期限后成为社会共同财富
    duration()   → "过期 → 公共领域"
}

// ⚠️ 注意区分：
// 地域性 vs 国际公约：国际公约简化跨国申请流程，
//   但最终保护仍依各国法律（地域性依然成立）
// 时间性：需精确记忆起止时间（如发明20年、著作权50年）
核心：知识产权是抽象的无形财产权，区别于有形财产。 四大特性中，无体性是本质特征，专有性是权利特征，地域性和时间性是保护特征。
📝 软件著作权（自动产生） 🔧 发明专利（保护期20年） 🏷️ 商标权（可续展，无固定期限） 🌐 国际公约（伯尔尼、TRIPS等）

✅ 掌握优势

  • 概念清晰，四个特性易记
  • 考试中属于送分题
  • 与生活实例关联紧密，好理解

⚠️ 易错点

  • 需区分地域性与国际公约的关系
  • 时间性的起止时间需精确记忆
  • 无体性 ≠ 不需要载体（载体≠权利本身）

📝 考试关键词

无体性 专有性 地域性 时间性 抽象财富
