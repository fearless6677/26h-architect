软件著作权保护规则

软件著作权自开发完成之日起产生，委托开发无合同归受托人，保护期自然人终生+50年/法人50年

🎬 权利产生与保护期
🔀 委托/合作开发
💻 法条速记
🖥️ 开发完成之日
著作权自动产生——无需登记、无需发表。完成即保护！
👤 自然人保护期
终生 + 死后50年（截止于第50年12月31日）
🏢 法人保护期
首次发表后50年（截止于第50年12月31日）。50年内未发表→不再保护
⚠️ 保护边界
不延及思想、处理过程、操作方法、数学概念——只保护表达，不保护思想

场景规则易错点
委托开发·有合同按合同约定—
委托开发·无合同著作权归受托人常考！不是委托人
合作开发·可分割各自享有各自部分—
合作开发·不可分割共同享有—
合作·协商不成不得阻止他方（转让除外）转让必须全体同意
// ═══ 软件著作权核心法条速记 ═══

// 1. 权利产生时点
著作权 = 开发完成之日自动产生    // ≠ 发表日 ≠ 登记日

// 2. 保护期
自然人: 终生 + 死后50年  (截止12月31日)
法人:   首次发表后50年  (截止12月31日, 50年未发表=不再保护)

// 3. 保护边界
保护(表达)    // 源代码、目标代码、文档
不保护(思想)  // 算法、处理方法、数学概念

// 4. 委托开发
if (有书面合同) → 按合同
else           → 归受托人     // 常考陷阱！

// 5. 合作开发
可分割 → 各自享有
不可分割 → 共同享有
协商不成 → 任何一方不得阻止他方行使(转让除外)
核心：软件著作权自开发完成之日起产生，不是登记也不是发表。 保护的是代码表达而非背后的思想/算法。委托开发无合同→归受托人（最常考的陷阱！）。
🖥️ 软件著作权自动产生（完成即保护） 📝 源程序 = 目标程序（同一作品） 🤝 委托无合同 → 受托人享有 👥 合作不可分割 + 协商不成 → 转让除外均可行使

✅ 学习优势

  • 练习题反复出现，考点高度重复
  • 法条简洁，易于掌握

❌ 易错提醒

  • 保护期的自然人/法人区别需分别记忆
  • 合作开发软件的著作权行使规则较复杂

📝 考试关键词
