// 诺兰6阶段模型 —— 评估组织信息化水平
enum NolanStage {
    ① 初始     // 刚引入计算机，零星使用
    ② 传播     // 各部门纷纷开发应用
    ③ 控制     // 管理层开始统筹控制
    ④ 集成     // 系统互联互通
    ⑤ 数据管理  // 以数据为核心统一管理
    ⑥ 成熟     // 信息系统全面融入业务
}

// 核心：渐进演化，不可跳跃（理论上）
// 用途：评估组织当前处于哪个阶段
// 提出者：Richard L. Nolan（理查德·诺兰）
核心：理查德·诺兰（Richard L. Nolan）提出信息系统发展的6阶段模型： ①初始期（刚引入计算机）→ ②传播期（各部门纷纷开发应用）→ ③控制期（管理层开始统筹控制）→ ④集成期（系统互联互通）→ ⑤数据管理期（以数据为核心统一管理）→ ⑥成熟期（信息系统全面融入业务）。 该模型用于评估组织信息化水平，是考试选择题高频考点。
初始期 🖥️ 公司刚买几台电脑打字
传播期 📦 各部门各自买软件
集成期 🔗 公司统一上ERP集成所有部门
📊 评估信息化水平 📈 渐进演化 🎯 考试高频考点 👤 Richard L. Nolan

✅ 优点

  • 为组织信息化水平提供评估框架
  • 帮助判断当前应投入的方向

❌ 缺点

  • 阶段划分较理想化
  • 不同行业可能跳跃式发展
