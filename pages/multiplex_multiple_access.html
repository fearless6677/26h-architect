// 复用与多址技术对比
// 复用技术: 一条信道 → 多路数据
  TDM(时分复用): 按时间片划分
  FDM(频分复用): 按频率划分
  CDM(码分复用): 按编码划分

// 多址技术: 一条线 → 多个用户数据
  TDMA(时分多址): 按时间区分用户
  FDMA(频分多址): 按频率区分用户
  CDMA(码分多址): 按编码区分用户

// 核心区别
// 复用 = 一条路上行驶多辆货车（多路数据共享信道）
// 多址 = 一辆车上的货物属于不同用户（多用户共享信道）
核心：复用技术是在一条信道上同时传输多路数据（如TDM/FDM/CDM）；多址技术是在一条线上同时传输多个用户数据并在接收端分离（如TDMA/FDMA/CDMA）。
比喻：复用 = 一条路上行驶多辆货车；多址 = 一辆车上的货物属于不同用户。
📡 TDM：将时间分为若干时隙，每路信号占用一个时隙 📶 CDMA：每个用户用不同的扩频码，在同一频率同时通信

✅ 优点

  • 提高信道利用率
  • 降低传输成本
  • 支持多用户并发通信

❌ 缺点

  • 需要复杂的同步机制
  • 存在干扰问题
