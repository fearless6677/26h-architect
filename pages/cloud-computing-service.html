云计算服务模式与部署模式

云计算三种服务方式（SaaS/PaaS/IaaS）和四种部署模式（公有云/社区云/私有云/混合云），注意'计算即服务'不是云计算服务方式（练习题常考干扰项）。

📖 核心解释

云计算是同时描述一个系统平台或一类应用程序的术语。三种服务方式：SaaS（软件即服务，提供商将应用软件统一部署在云端，用户直接使用）、PaaS（平台即服务，提供商将分布式开发环境与平台作为服务提供）、IaaS（基础设施即服务，提供商将服务器组成云端基础设施按计量供给）。四种部署模式：公有云（面向公众开放）、社区云（特定社区共享）、私有云（单一组织专用）、混合云（两种或以上云组合）。注意：云计算服务方式不包括'计算即服务'，这是常见干扰项。

💻 伪代码

// 云计算三种服务方式
服务模式 = {
  "SaaS": "软件即服务→应用软件统一部署在云端（用户直接使用应用）",
  "PaaS": "平台即服务→分布式开发环境与平台作为服务（开发者使用平台）",
  "IaaS": "基础设施即服务→服务器组成'云端'基础设施按量计费（管理员使用基础设施）"
}

// 四种部署模式
部署模式 = ["公有云", "社区云", "私有云", "混合云"]

📌 实际例子

  • SaaS：Office 365、Salesforce等在线软件服务
  • PaaS：Google App Engine、AWS Elastic Beanstalk等开发平台
  • IaaS：阿里云ECS、AWS EC2等按需计算资源
  • 混合云：企业核心数据放私有云，弹性业务用公有云

✅ 优点

  • 按需使用、弹性伸缩
  • 降低IT基础设施投入成本
  • 快速部署、运维简化
