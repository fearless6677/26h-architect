嵌入式微处理器五大类型

嵌入式处理器五大家族：MPU（裸处理器）、MCU（单片机集成一切）、DSP（数据运算专家）、GPU（并行计算）、SoC（全系统集成一芯片）。

集成度：MPU < MCU < DSP < GPU < SoC

// 五类处理器对比
处理器类型 = {
  MPU:  微处理器+外接电路板, 如ARM/PowerPC/MIPS,
  MCU:  单片机(片内集成存储+外设), 如STM32/8051,
  DSP:  哈佛结构, 擅长大数据运算, 如TMS320系列,
  GPU:  浮点+多核并行, 用于AI深度学习,
  SoC:  完整系统集成单芯片(处理器+IP核+存储+OS)
}
// 存储速度排序
速度: 寄存器组 > Cache(SRAM) > 内存(DRAM) > Flash > ROM
核心：嵌入式微处理器分五类：MPU（微处理器，需外接电路板，如ARM/PowerPC/MIPS）、MCU（单片机，存储+外设集成片内，单片化体积小功耗低）、DSP（数字信号处理器，哈佛结构，擅长大数据量运算）、GPU（图形处理器，浮点+多核并行，用于AI深度学习）、SoC（片上系统，处理器+IP核+存储+OS全集成）。
MCU应用：STM32单片机控制智能家居设备，单芯片完成传感+控制+通信 SoC应用：手机芯片骁龙888，CPU+GPU+NPU+ISP全部集成在一块芯片上 DSP应用：音频处理芯片对声音信号进行实时傅里叶变换

✅ 优点

  • MCU单片化体积小成本低，适合嵌入式场景
  • SoC集成度高，整体性能最优
  • DSP对数字信号处理效率远超通用CPU

❌ 缺点

  • MPU集成度低，需外围电路设计
  • DSP功耗较大，不适合电池供电场景
