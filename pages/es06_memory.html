// 存储器分类及特点
RAM(易失):                      // 断电数据丢失，需供电维持
  DRAM: 电容存储 → 需刷新       // 电容漏电 → 定期刷新充放电
       → 集成度高, 容量大, 速度慢  // 结构简单 → 单位面积容量大
       → 用作主存                 // PC内存条 = DDR SDRAM

  SRAM: 晶体管自锁 → 不需刷新   // 6个晶体管组成触发器，状态自保持
       → 速度快, 集成度低, 成本高  // 6T单元 vs 1T1C → 面积大
       → 用作Cache               // CPU内部 L1/L2/L3 高速缓存

ROM(非易失):                    // 断电数据不丢失
  MROM:  掩膜制造 → 大批量, 不可修改   // 出厂内容固定，成本最低
  PROM:  一次性烧录 → 少量制造         // 熔丝原理，写错就废
  EPROM: 紫外线擦除重写               // 芯片上有石英窗口
  EEPROM: 电压擦除 → 速度慢           // 按字节擦写，寿命长
  Flash:  联机擦写 → 速度快           // 按块擦写 → U盘/SSD/固件存储
              次数多, 读速慢            // 擦写寿命 ~10万次
类型 存储原理 易失性 可改写 典型用途
DRAM 电容充放电 易失 ✅ 可读写 PC/嵌入式主存（内存条）
SRAM 6T触发器自锁 易失 ✅ 可读写 CPU内部 L1/L2 Cache
MROM 掩膜光刻 非易失 ❌ 不可改 大批量固化程序（BIOS早期）
PROM 熔丝烧断 非易失 ❌ 一次性 少量定制化固件
EPROM 浮栅晶体管 非易失 ⚠️ 紫外线擦除 早期开发调试（石英窗口）
EEPROM 浮栅隧道效应 非易失 ✅ 电压擦除 设备配置参数（按字节写）
Flash 浮栅/电荷俘获 非易失 ✅ 联机擦写 固件/U盘/SSD（按块写）
核心：存储器分两大类——RAM（易失性，断电丢数据）和ROM（非易失性，断电保数据）。 RAM 分 DRAM（电容存储，需刷新，集成高容量大，用于主存）和 SRAM（晶体管自锁，不需刷新，速度快集成低，用于Cache）。 ROM 演化出 5种：MROM → PROM → EPROM → EEPROM → Flash，擦写越来越方便。 存取速度排序：寄存器组 ＞ Cache ＞ 内存(DRAM) ＞ Flash。
💻 DRAM → PC和嵌入式系统主存 ⚡ SRAM → CPU内部L1/L2高速缓存 📦 Flash → 固件存储和U盘/SSD ⚙️ EEPROM → 存储设备配置参数

✅ 层次化存储的优势

  • 平衡性能与成本：高速小容量(Cache) + 低速大容量(主存/Flash)
  • 局部性原理：时间局部性 → Cache命中；空间局部性 → 块读取
  • 对程序员透明：硬件自动管理Cache，无需手动干预
  • ROM演化灵活：从不可改(MROM)到联机擦写(Flash)，适应不同阶段

⚠️ 注意事项

  • DRAM需定时刷新，否则电容漏电丢失数据
  • Flash有擦写寿命限制（约10万次），写磨损均衡很关键
  • EEPROM按字节写速度慢，大批量写入选Flash
  • SRAM成本高集成度低，不能无限增大Cache
