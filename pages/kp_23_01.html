知识产权四大特性

知识产权具有无体性、专有性、地域性、时间性四大特征

🎬 动画演示
💻 代码速记

四大特性构成知识产权的完整框架——缺一不可

💭
无体性
Intangibility
对象没有具体形体
是抽象的智力成果
💡 一首歌的著作权——看不见摸不着，但它是财富
🔒
专有性
Exclusivity
未经许可他人不得使用
权利人独占享有
💡 未经作者许可不得复制其作品
🗺️
地域性
Territoriality
只在授予权利的国家
范围内受保护
💡 一项中国专利在美国不受保护
⏳
时间性
Temporality
超过法定期限后
成为社会共同财富
💡 著作权财产权保护期满后作品进入公有领域
// 知识产权四大特性 —— 一切IP制度的基石
const IP特性 = {
  无体性: {
    // 对象没有具体形体，是抽象财富
    对象: "智力成果",        // 不是实物！
    类比: "一首歌 ≠ 一张CD"
  },
  专有性: {
    // 未经许可，他人不得使用
    核心: "独占 + 排他",
    例外: "权利人同意 或 法律允许"
  },
  地域性: {
    // 只在授予权利的国家有效
    范围: "国境线内",
    例子: "中国专利 → 美国无效"
  },
  时间性: {
    // 超过法定期限 → 社会共同财富
    期满: "进入公有领域",
    口诀: "有期限 → 终会到期"
  }
}
// 🧠 速记：无体专有 + 地域时间
核心：知识产权有四大特征—— 无体性（对象是抽象成果）、专有性（未经许可不得使用）、 地域性（仅在授权国有效）、时间性（超过期限成公共财富）。 这四个特性是理解所有知识产权制度的基础。
💭 无体性：抽象智力成果 🔒 专有性：独占排他 🗺️ 地域性：国境内有效 ⏳ 时间性：有期限保护

✅ 学习优势

  • 概念清晰，容易理解和记忆
  • 是理解后续各知识产权法律的基础
  • 四个特性框架完整，逻辑自洽

❌ 易错点

  • 需要记忆四个特性的具体含义，容易混淆
  • 注意区分"专有性"与"地域性"的适用场景
  • 注意"时间性"中不同IP的保护期限差异

📝 考试关键词 约1分

无体性 专有性 地域性 时间性 知识产权特征
典型考法：
  • 一项中国专利在美国不受保护 → 地域性
  • 著作权财产权保护期满后作品进入公有领域 → 时间性
  • 未经作者许可不得复制其作品 → 专有性
