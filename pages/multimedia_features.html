多媒体4大特征与关键技术

多媒体4大特征：多维化、集成性、交互性、实时性；关键技术含数据压缩（有损/无损）和VR/AR技术。

📖 核心解释

多媒体有4大特征：多维化（媒体多样化）、集成性（设备/信息/表现集成）、交互性（用户可控制信息）、实时性（音视频有时间特性）。关键技术包括：视音频技术（数字化+编解码）、通信技术、数据压缩技术（有损/无损、即时/非即时）、VR/AR技术（桌面式/分布式/沉浸式/增强式4种）。VR创建虚拟世界，AR将虚拟信息叠加到现实世界。

💻 伪代码

// 多媒体4大特征
多维化  → 媒体形式多样（文字/声音/图像/动画/视频）
集成性  → 多媒体与设备集成、与信息和表现集成
交互性  → 用户可主动控制和选择信息
实时性  → 音频/视频有强时间特性，必须同步处理

// 数据压缩
有损压缩 → 丢弃部分信息，压缩比高（JPEG/MP3）
无损压缩 → 完整保留，压缩比低（ZIP/PNG）
即时压缩 → 实时处理
非即时压缩 → 离线处理

📌 实际例子

  • JPEG是有损压缩，PNG是无损压缩
  • VR用于游戏/模拟训练，AR用于导航/教育
  • 视频会议需同时满足实时性和交互性

✅ 优点

  • 提供丰富的信息表现手段
  • 交互性提高用户体验
  • VR/AR创造沉浸式体验

❌ 缺点

  • 多媒体数据量大，对存储和带宽要求高
  • 实时性要求使得系统设计复杂
