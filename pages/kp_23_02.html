著作权归属与保护期

著作权人身权中署名权、修改权、保护作品完整权保护期不受限制；职务作品与委托作品归属是高频考点

🎬 动画演示
💻 法条速查
著作权
🛡️ 人身权
发表权 署名权 ★ 修改权 ★ 保护作品完整权 ★
保护期：署名权、修改权、保护作品完整权 → 不受限制（永久）
发表权：与财产权同期 → 作者终生 + 死后50年
💰 财产权（17项）
复制权 发行权 出租权 展览权 表演权 信息网络传播权 …等共17项
保护期：作者终生 + 死后50年（截止于作者死亡后第50年的12月31日）
法人作品：首次发表后50年
🏢 职务作品
📜 委托作品
📖 合理使用
一般：著作权归作者
特殊：主要利用单位条件+单位承担责任 → 作者仅享署名权，其余归单位
有合同：按合同约定
无合同：著作权归受托人
第24条：不经许可、不付报酬
须：指明作者+出处
例：个人学习、评论引用 
// ═══ 著作权归属与保护期 · 法条速查 ═══

// ── 一、人身权（4项）──────────────
人身权 = { 发表权, 署名权, 修改权, 保护作品完整权 }

// ★ 高频考点
保护期(署名权)           → 不受限制（永久）
保护期(修改权)           → 不受限制（永久）
保护期(保护作品完整权)   → 不受限制（永久）
保护期(发表权)           → 与财产权相同（终生+50年）

// ── 二、财产权（17项）─────────────
财产权 = { 复制权, 发行权, 出租权, 展览权,
          表演权, 放映权, 广播权, 信息网络传播权,
          摄制权, 改编权, 翻译权, 汇编权, … }

保护期(财产权) → 自然人: 作者终生 + 死后50年
                 法人:   首次发表后50年

// ── 三、职务作品 ──────────────────
if (主要利用单位物质技术条件 && 单位承担责任) {
    // 特殊职务作品（如软件、工程设计图）
    作者享有 → 署名权     // 仅这一项！
    其他权利 → 归单位
} else {
    // 一般职务作品
    著作权 → 归作者（单位有优先使用权）
}

// ── 四、委托作品 ──────────────────
if (有合同约定) {
    著作权 → 按合同
} else {
    著作权 → 归受托人  // ★ 无合同 = 归受托方
}

// ── 五、合理使用（第24条）──────────
合理使用 → 不经许可 + 不付报酬 + 须指明作者/出处
// 常见情形：个人学习、评论引用、新闻报道、课堂教学等
核心：著作权分人身权和财产权两大类。 人身权中署名权、修改权、保护作品完整权保护期不受限制（永久保护）。 财产权保护期为作者终生+死后50年。
归属规则：职务作品特殊情况下作者仅保留署名权；委托作品无合同约定归受托人。 合理使用（第24条）允许不经许可、不付报酬使用已发表作品。
📝 署名权/修改权/完整权 → 永久保护 📜 委托开发无合同 → 归受托方 🏢 员工在职开发软件 → 职务作品归单位 📖 个人学习使用 → 合理使用

✅ 备考优势

  • 考点明确，出题规律稳定
  • 法条直接对应选择题
  • 规则清晰，容易记忆对比

❌ 易混淆点

  • 权利类型众多（17项财产权），容易混淆人身权和财产权
  • 保护期规则因权利类型不同而不同
  • 职务作品"一般"与"特殊"归属易混

📝 考试关键词

著作权 人身权 财产权 保护期 职务作品 委托作品 合理使用 署名权不受限制

📊 分值权重：约1-2分（高频考点）

🔗 关联知识

  • → 专利权 — 与著作权同属知识产权，保护期规则不同
  • → 商标权 — 知识产权三大支柱之一，续展规则不同
  • 对比记忆：著作权自动取得（无需登记） vs 专利权/商标权需申请审批
  • 软件保护条例：软件著作权自开发完成之日起产生，保护期25年（可续展）
