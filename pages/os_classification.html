// 操作系统分类速记
批处理OS  → 作业批量提交，无交互（单道/多道）
分时OS    → CPU时间片轮转，多路性/独立性/交互性/及时性
实时OS    → 快速响应，高可靠性，不强制交互（VxWorks等）
网络OS    → 共享网络资源，硬件独立性，多用户
分布式OS  → 网络OS的高级形式，透明性/可靠性/高性能
嵌入式OS  → 微型化/可定制/易移植（HAL+BSP）

// OS 四大特征 = 并发性 + 共享性 + 虚拟性 + 不确定性
核心：操作系统按类型分为6类：①批处理OS（作业批量执行，无交互）；②分时OS（时间片轮转，具多路性/独立性/交互性/及时性4大特点）；③实时OS（快速响应，高可靠性，如VxWorks、μC/OS-II）；④网络OS（共享网络资源）；⑤分布式OS（网络OS的高级形式，有透明性）；⑥嵌入式OS（微型化、可定制，采用HAL和BSP提高可移植性）。OS具有并发性、共享性、虚拟性、不确定性4大特征。
🖥️ UNIX/Linux 多用户终端（分时OS） 🚀 VxWorks 火星车（实时OS） 📱 Windows CE / eCos（嵌入式OS）

✅ 优点

  • 分时OS：多用户交互体验好
  • 实时OS：响应及时、可靠性极高
  • 嵌入式OS：微型化、可裁剪，适合资源受限设备

❌ 局限

  • 批处理OS：无交互能力，调试困难
  • 实时OS：成本高，开发复杂度大
  • 嵌入式OS：硬件依赖强，跨平台移植有挑战

📝 考试关键词
