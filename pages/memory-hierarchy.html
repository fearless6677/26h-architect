存储器层次结构

计算机存储器分为寄存器、缓存、内存、外存等多个层次，越快越小越贵，越慢越大越便宜。

// 存储器层次结构（由快到慢、由小到大）
L0: CPU寄存器   // 最快，最小，KB级
L1: 片上缓存    // Cache（SRAM），很快，KB~MB级
L2: 片外缓存    // 二级缓存，MB级
L3: 主存(内存)  // DRAM，较慢，GB级
L4: 外存(磁盘)  // 最慢，最大，TB级
// 原则：越快越贵越小，越慢越大越便宜
核心：计算机存储器分为寄存器、缓存、内存、外存等多个层次，越快越小越贵，越慢越大越便宜。
CPU中的L1/L2 Cache使用SRAM，速度极快但容量仅几MB，用于缓存频繁访问的数据 电脑内存条使用DRAM（如DDR4/DDR5），容量16GB，断电数据丢失 硬盘（HDD/SSD）作为外存，容量可达TB级，用于长期保存操作系统和文件

✅ 优点

  • 利用局部性原理，用少量高速缓存模拟大容量存储器，性价比极高
  • 层次化设计使系统整体性能接近最快存储器，容量接近最慢存储器
  • 各级存储技术可以独立发展和替换，扩展性好

❌ 缺点

  • 高速存储器（SRAM/Cache）成本高，容量不可能无限扩大
  • 层次越多，缓存一致性、数据同步问题越复杂
  • 外存速度远慢于CPU，I/O瓶颈长期存在
