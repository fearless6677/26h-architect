# 26小时通关系统架构师 · 知识可视化学习系统

一个基于纯 HTML 的系统架构设计师考试学习系统，26小时通关 236 个核心知识点，包含 267 个交互页面，支持思维导图动画、知识卡片、视频演示等互动学习功能。

## ✨ 特性

- 🧠 **知识地图**：交互式思维导图，支持全屏、搜索、折叠展开
- 📚 **267 个知识点页面**：每个知识点独立讲解，结构清晰
- 🎬 **架构动画视频**：管道-过滤器、黑板模式、事件驱动等架构模式动态演示
- 🎯 **考点标记**：热点考点、普通考点权重标记，重点一目了然
- 📱 **响应式设计**：PC、平板、手机完美适配
- ⚡ **纯前端**：零后端依赖，可离线使用，可部署到任何静态托管平台

## 🚀 快速开始

### 在线访问

**GitHub Pages**: [https://fearless6677.github.io/26h-architect/](https://fearless6677.github.io/26h-architect/)

```bash
# 克隆仓库
git clone https://github.com/fearless6677/26h-architect.git
cd 26h-architect

# 启动本地服务器
python -m http.server 8000
# 或
npx serve

# 浏览器访问 http://localhost:8000
```

## 🤖 AI 互动学习

本项目支持 AI 辅助学习，详见 [CLAUDE.md](CLAUDE.md)。

**使用方式**：
1. 打开任意知识点页面
2. 向 AI 提问："详细解释这个页面的内容"、"这个知识点怎么考"、"给我举个实际案例"
3. AI 基于页面内容进行深入讲解、真题分析、案例扩展
